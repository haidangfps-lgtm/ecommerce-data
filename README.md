## 📌 Project Overview
This project focuses on extracting, cleaning, and visualizing e-commerce data from top platforms in Vietnam (Shopee, Tiki, Lazada). The goal is to process raw sales data and uncover actionable business insights regarding market share, revenue trends, and top-selling product categories.

## 🛠️ Technologies & Tools Used
* **Data Processing (Data Wrangling):** Python (Pandas, NumPy)
* **Data Visualization:** Power BI
* **Version Control:** Git & GitHub

## 🚀 Architecture & Workflow
1. **Data Extraction:** Collected raw sales datasets across 3 major e-commerce platforms (Shopee, Tiki, Lazada).
2. **Data Transformation (Data Cleaning):** Developed Python scripts using Pandas and NumPy to clean messy data, handle missing values, format data types, and consolidate datasets into a unified structure.
3. **Data Visualization:** Imported the cleaned datasets into **Power BI** to build interactive dashboards for tracking KPIs and business metrics.
